Lsystem-Compiler

Compiles Lsystem source code into Java bytecode.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/ethanhann/Lsystem-Compiler

How to Compile

Compiling and install the Lsystem compiler is easy. You need make and O'Caml installed, then... 

$ cd Lsystem-Compiler
$ make
$ cp _build/lsystem /usr/bin

Compiling an Lsystem program is easy. You need the Java compiler installed, then... 

$ lsystem -c foo.ls

Running an Lsystem program is easy too. You need Java installed, then... 

$ java foo

def draw dragonCurve(int level){
    alphabet:  (X,Y,Z);
    rules:{
        lambda -> Z;
        Z -> f X;
        X -> X l Y f;
        Y -> f X r Y;
    }
}
